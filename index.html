Website Design

  • Utilized Figma to generate wireframes
  • Implemented layout using CSS Flexbox & Grid
  • Employed SCSS functions and mixins
  • Designed with a mobile-first approach
  • Achieved responsive web design with 8 breakpoints and responsive text

Website Features

Menu

  • Small screens: Accordions and carousel with a progress bar
  • Large screens: Menu tabs with panels

Reservation

  • Reservation form with default values (date and time)
  • Initial form validation on the client side
  • Pop-up message to confirm booking details

Quality Assurance

  • Achieved near-perfect scores on Google PageSpeed Insights, consistently scoring 96-100 for performance, accessibility, best practices, and SEO on both mobile and desktop platforms
  • Website loads quickly, is accessible to people with disabilities, and is easily searchable by search engines
  • All HTML and CSS validated by W3C
  • Cross-browser compatible ensuring consistent performance and appearance across various web browsers

Accessibility

  • Design with sufficient color contrast
  • Skip link for easy navigation
  • Sufficient ARIA tags for screen readers
  • Optimized for keyboard users

Streamlined Workflow

Utilized Gulp and npm to:

  • Generate development and production code
  • Resize and optimize images
  • Convert font format from TIFF to WOFF2
  • Use BrowserSync to update changes live
  • Create minified styles from CSS files automatically
  • Create minified critical and non-critical styles from SCSS files automatically
  • Create minified critical and non-critical scripts in legacy and modern code automatically
